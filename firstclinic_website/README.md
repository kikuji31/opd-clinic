# First Clinic — เว็บไซต์ (20 หน้า)

เว็บไซต์คลินิกความงาม "First Clinic" โทนสีขาว-ทอง สร้างตามผังเว็บใน Google Sheet ที่ให้มา
และเนื้อหาอ้างอิงจาก [firstclinic.co.th](http://firstclinic.co.th/) — First Clinic เป็นคลินิก
เฉพาะทางฟิลเลอร์รอบดวงตา ดูแลโดยจักษุแพทย์ (นพ.ลัทธพล ม้าลายทอง) จึงปรับหมวดบริการในผังเว็บต้นฉบับ
(ที่เขียนแบบคลินิกทั่วไปว่ามีโบท็อกซ์/เลเซอร์) ให้ตรงกับบริการจริงที่คลินิกมี คือ ฟิลเลอร์ (หลายจุด),
Ulthera, Meso Therapy, ผ่าตัดถุงใต้ตาแบบไม่เปิดแผล และแว่นตา BE1st

## ดูตัวอย่างเว็บตอนนี้

เปิด `index.html` ในเบราว์เซอร์ได้ทันที ทุกหน้าลิงก์ถึงกันและใช้ `assets/style.css` +
`assets/script.js` ร่วมกัน — ไม่ต้องใช้ server

## ⚠️ ข้อมูลที่ยังเป็น Placeholder ต้องแก้ก่อนเผยแพร่จริง

เว็บนี้สร้างจากข้อมูลสาธารณะของ First Clinic เท่าที่หาได้จากเว็บไซต์จริงเท่านั้น
**สิ่งต่อไปนี้ยังไม่มีข้อมูลจริง ห้ามเผยแพร่โดยไม่แก้ไขก่อน:**

| รายการ | อยู่ในหน้า | หมายเหตุ |
|---|---|---|
| เลขที่ใบอนุญาตสถานพยาบาล | about.html | จำเป็นตามกฎหมาย |
| เลขที่ใบประกอบวิชาชีพเวชกรรมของแพทย์ | doctors.html, doctor-lattapol.html | จำเป็นตามกฎหมาย (YMYL) |
| ที่อยู่สาขา / เวลาทำการจริง | contact.html, locations.html, footer ทุกหน้า | ใส่ก่อนเปิดเว็บจริง |
| แผนที่ Google Maps จริง | contact.html, locations.html, index.html | ตอนนี้เป็นกล่อง placeholder |
| ลิงก์ LINE OA / Facebook / YouTube จริง | footer ทุกหน้า, ปุ่มลอย LINE | ตอนนี้เป็น `href="#"` หรือ URL ทั่วไป |
| ภาพถ่ายจริง (คลินิก, แพทย์, ก่อน-หลัง) | ทุกหน้าที่มีกล่อง "ตัวอย่าง/[ใส่รูปจริง]" | ต้องมีหนังสือยินยอมก่อนเผยแพร่ภาพเคสจริง |
| รีวิว/เคสจริง | reviews.html, index.html | ตอนนี้เป็นข้อความตัวอย่างที่แต่งขึ้น มีป้าย "ตัวอย่างรีวิว" กำกับ |
| ราคาโปรโมชั่นจริง + เงื่อนไข | promotions.html | ตอนนี้เป็นราคาตัวอย่าง มีป้าย "ตัวอย่างโปรโมชั่น" กำกับ |
| บทความจริง | blog.html | ตอนนี้มี 1 บทความตัวอย่างเต็ม + การ์ดตัวอย่างอีก 6 |
| อีเมล/เบอร์โทร DPO | privacy.html | จำเป็นตาม พ.ร.บ. คุ้มครองข้อมูลส่วนบุคคล |
| ข้อความกฎหมาย (privacy/terms) | privacy.html, terms.html | เป็นร่างเบื้องต้น ควรให้ทนายความตรวจก่อนใช้จริง |

## วิธีนำขึ้น WordPress

ไฟล์ชุดนี้ทำเป็น HTML แยกไฟล์ต่อหน้าตามที่ขอ วิธีย้ายขึ้น WordPress ที่แนะนำ:

1. **CSS ใส่ครั้งเดียว** — คัดลอกเนื้อหาทั้งหมดใน `assets/style.css` ไปวางที่
   **Appearance → Customize → Additional CSS** (หรือถ้าใช้ปลั๊กอิน เช่น WPCode/Simple Custom CSS
   ก็ได้เช่นกัน) ทำครั้งเดียว มีผลกับทุกหน้า ไม่ต้องคัดลอกซ้ำในแต่ละเพจ
2. **JS ใส่ครั้งเดียว** — วางเนื้อหาใน `assets/script.js` ผ่านปลั๊กอิน "Insert Headers and
   Footers" หรือ WPCode โดยห่อด้วย `<script>...</script>` วางใน footer ของทุกหน้า (ทำครั้งเดียว
   เช่นกัน) ต้องมี Google Fonts link (Anuphan, Inter) ด้วย — ใส่ผ่าน header snippet หรือปลั๊กอิน
   Google Fonts ก็ได้
3. **แต่ละหน้า → สร้างเป็น WordPress Page ใหม่** — สร้างเพจใหม่ตามชื่อหน้า (เช่น "เกี่ยวกับเรา"),
   ใช้บล็อก **Custom HTML** (หรือ Elementor → HTML widget) แล้ววางเฉพาะส่วนใน `<body>...</body>`
   ของไฟล์นั้น (ไม่ต้องเอา `<head>`, ไม่ต้องเอาแท็ก `<link rel="stylesheet">`/`<script src=...>`
   เพราะ CSS/JS ใส่ไว้ที่ Additional CSS/footer แล้วในขั้นตอนที่ 1-2)
4. **ตั้ง URL slug ให้ตรงกับลิงก์ภายใน** — เพจแต่ละอันควรตั้ง slug ให้ตรงกับชื่อไฟล์ (เช่นเพจ
   "เกี่ยวกับเรา" ตั้ง slug เป็น `about`) เพื่อให้ลิงก์เมนู/ปุ่มต่างๆ ในเนื้อหาที่คัดลอกไปยังชี้ถูกที่
   หรือจะแก้ `href="about.html"` เป็น `href="/about/"` (หรือ URL จริงตาม permalink ของ WordPress)
   ในเนื้อหาก่อนวางก็ได้เช่นกัน
5. **เมนูนำทาง** — เมนู/dropdown/hamburger ที่อยู่ในทุกหน้าเป็นส่วนหนึ่งของ Custom HTML อยู่แล้ว
   ใช้งานได้ทันทีโดยไม่ต้องตั้งค่า WordPress Menu เพิ่ม (แต่ถ้าธีมมี header ของตัวเองอยู่แล้ว
   อาจต้องเลือกใช้ page template แบบ "Full Width / No Header" เพื่อไม่ให้ nav ซ้อนกัน)

## Compliance ที่ทำไว้ตามผังเว็บต้นฉบับ

- หลีกเลี่ยงคำโฆษณาเกินจริง ("ดีที่สุด", "หายขาด 100%") ทุกหน้าใช้ภาษาที่วัดผลได้/ระบุว่าแตกต่างกันในแต่ละบุคคล
- ทุกหน้าบริการจบด้วย CTA แบบ "ปรึกษา/นัดหมาย" ไม่ใช่ "ซื้อเลย"
- `faq.html` มี FAQPage schema (JSON-LD) พร้อมใช้สำหรับ SEO/AEO
- `contact.html` มี checkbox ยินยอม PDPA ผูกกับ `privacy.html`
- `reviews.html` และ `promotions.html` ติดป้าย "ตัวอย่าง" ชัดเจนทุกจุดที่เป็นข้อมูลสมมติ
